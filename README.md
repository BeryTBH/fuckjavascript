# fuckjavascript

fuckjavascript obfuscates javascript code and makes your code harder to read.

## How to Use

- Have all the requiements checked for this to work
- Open a cmd window in the folder and run this: ```npm install```, once done you need to run ```node index.js <script>``` or instead run this command: ```npm run obf <script>```
- Obfuscated script should be in out.js, congrats if it worked for you, if it didnt, post a [issue](https://github.com/BeryTBH/fuckjavascript/issues).

## Requirements

- NodeJS 21 or above
- Have a brain
